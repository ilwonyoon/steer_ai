# README screenshots

The main `README.md` references three screenshots. Drop real PNGs at these
exact paths and the page fills in.

| Path | What to capture |
|---|---|
| `hero.png` | The Mac app showing a card stack with at least 2 stopped sessions visible. ~1800x1000, light background. This is the top banner — make it the most "this is what Steer looks like" shot you have. |
| `mac-card-stack.png` | Same Mac window, tighter framing on the card stack itself — terminal-tail excerpt should be legible. ~1200x800. |
| `mac-reply.png` | Card detail with the reply dock focused and a typed reply visible. ~1200x800. |

## Capture

- macOS Tahoe: `Cmd+Shift+5` → **Capture Selected Window**.
- Hold `Option` while clicking to skip the drop shadow.
- Save as PNG, no compression.

## Rules

- Real screenshots only. No mocks, no rendered previews — they drift from the actual app.
- Use a real `steer claude` or `steer codex` session that's genuinely stopped, not a fixture.
